<h1 align="center">AASTU Gebeya</h1>

<p align="center">
  Campus E-Commerce Platform Built with MERN Stack
</p>

AASTU Gebeya is a full-stack MERN web application designed to simplify and modernize campus e-commerce within Addis Ababa Science and Technology University (AASTU).  
The platform enables students to easily buy, sell, and explore products inside the university community through a centralized digital marketplace.

---

# Features

- User authentication system
- Buyer and seller accounts
- Product listing and management
- Product detail pages
- Explore and browse products
- Responsive modern UI
- Campus-focused marketplace experience
- Easy item posting for students
- Profile management system

---

#  Tech Stack

## Frontend
- React
- Vite
- CSS3
- JavaScript (ES6+)

## Backend
- Node.js
- Express.js

## Database
- MongoDB

#  Installation

## 1. Clone the Repository

```bash
git clone https://github.com/Hayat-Musema/AASTU-Gebeya.git
```

---

## 2. Navigate to Project Folder

```bash
cd AASTU-Gebeya
```

---

#  Frontend Setup

```bash
cd Frontend/AASTU-Gebeya-Frontend
npm install
npm run dev
```

Frontend will run on:

```bash
http://localhost:5173
```

---

#  Backend Setup

```bash
cd Backend
npm install
npm start
```

---

#  Environment Variables

Create a `.env` file inside the backend folder and configure:

```env
MONGO_URI=your_mongodb_connection
PORT=5000
JWT_SECRET=your_secret_key
```

---

#  Project Goal

The main goal of AASTU Gebeya is to create a trusted digital marketplace for university students where they can easily exchange products and services within campus.

The platform aims to:
- simplify student-to-student trading
- reduce difficulties in finding buyers and sellers
- encourage digital commerce culture inside campus
- provide a secure and accessible marketplace experience

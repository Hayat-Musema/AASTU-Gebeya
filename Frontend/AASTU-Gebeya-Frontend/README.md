# AASTU Gebeya Frontend

Frontend of the AASTU Gebeya platform built using React and Vite.

## 🚀 Tech Stack

- React
- Vite
- CSS3
- JavaScript (ES6+)

## 📂 Project Structure

```bash
src/
 ├── assets/
 ├── components/
 ├── App.jsx
 ├── main.jsx
 └── styles/
```

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Hayat-Musema/AASTU-Gebeya.git
```

Navigate to the frontend directory:

```bash
cd Frontend/AASTU-Gebeya-Frontend
```

Install dependencies:

```bash
npm install
```

## ▶️ Run Development Server

```bash
npm run dev
```

The application will run locally on:

```bash
http://localhost:5173
```

## 📦 Build for Production

```bash
npm run build
```

- Responsive modern UI

- Product details page
- Explore page
- User profile page
